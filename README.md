# Wanted Front-End Internship Pre-Onboarding

### 지원자

- 김명준

### 프로젝트 실행 방법

- 설치

```
npm install
```

- Dev 모드 실행

```
npm run start
```

- 프로젝트 빌드

```
npm run build
```
